
## @aws-sdk/client-ecs
### 1. @aws-sdk/client-ecs là gì?
Đây là bộ thư viện chính thức của Amazon Web Services (AWS) dùng để tương tác với dịch vụ **Amazon Elastic Container Service (ECS)** thông qua mã nguồn JavaScript/TypeScript.

### 2. Nó giải quyết vấn đề gì?
Khi ứng dụng của bạn cần tự động điều khiển các container (như Docker) trên đám mây (ví dụ: tự bật thêm server, tắt server, hoặc kiểm tra tình trạng ứng dụng), bạn cần thư viện này để "nói chuyện" với AWS.

### 3. Các tính năng chính:
*   **Thiết kế Modul (v3):** Bạn chỉ cần cài đúng phần mình cần (ECS), giúp giảm dung lượng dự án.
*   **Hỗ trợ TypeScript:** Cung cấp định nghĩa kiểu đầy đủ, giúp viết code chính xác và ít lỗi hơn.

### 4. Cách sử dụng phổ biến
Dùng trong backend hoặc các kịch bản tự động hóa để quản lý hạ tầng server đám mây.

**Tóm lại:** Đây là "cánh tay nối dài" giúp code của bạn điều khiển được các server container trên AWS.
